# Political Finance, EMB Oversight, and Corruption

Replication code for a panel data study examining whether public financing of
political parties reduces corruption — and whether this effect is conditional
on the capacity and autonomy of electoral management bodies (EMBs).

## Research question

Does public funding of political parties reduce corruption? And does the answer
depend on whether the electoral management body responsible for oversight
actually has the capacity to enforce the rules?

## Methods

- Two-way fixed effects panel models (country + year FE)
- Driscoll-Kraay standard errors (robust to serial correlation and cross-sectional dependence)
- Interaction models: public funding × EMB capacity/autonomy
- Event study with leads and lags (±3 years) to test for reverse causality
- Comprehensive specification analysis across 729 model variants

## Key variables

| Variable | Description | Source |
|---|---|---|
| `v2elpubfin` | Public funding of political parties | V-Dem v15 |
| `v2elembcap` | EMB capacity to oversee party finance | V-Dem v15 |
| `v2elembaut` | EMB autonomy from government | V-Dem v15 |
| `v2x_corr` | Political corruption index (outcome) | V-Dem v15 |
| `idea_narrow` | EMB held oversight responsibility in 2003 and 2022 | IDEA |
| `idea_broad` | EMB held oversight responsibility in 2022 | IDEA |

## Data

Two datasets are required (not included in this repository):

**V-Dem v15** — Varieties of Democracy country-year dataset  
Download: https://v-dem.net/data/the-v-dem-dataset/  
File: `V-Dem-CY-Full+Others-v15.csv` → place in `data/`

**IDEA Political Finance Database** — EMB oversight responsibility coding  
Source: https://www.idea.int/data-tools/data/political-finance-database  
(Manual coding of oversight responsibility in 2003 and 2022 is embedded in the script)

## Running the code

1. Clone this repository
2. Download V-Dem v15 and place in `data/`
3. Open `analysis.R` and run from top to bottom

The script is structured in numbered sections (1–12) and can be run
section by section or in full.
