# Gender-Targeted Public Funding and Women's Parliamentary Representation

Replication code for a panel data study examining whether gender-targeted
public funding (GTPF) of political parties increases women's representation
in parliament, across 158 countries and 1,183 elections.

## Research question

Does public funding of political parties increase women's parliamentary
representation? And does it matter *how* the funding is structured — through
allocation rules, eligibility conditions, or earmarking requirements?

## Methods

- Two-way fixed effects panel models (country + year FE)
- Driscoll-Kraay standard errors (robust to serial correlation and cross-sectional dependence)
- Four GTPF treatment variables: general GTPF, allocation, eligibility, earmarking
- Robustness checks: lagged treatment, alternative DV transformations, random effects, fractional logit, first differences, temporal persistence (1 and 2 elections ahead)
- IV estimation using party system institutionalisation as instrument

## Key variables

| Variable | Description | Source |
|---|---|---|
| `fem.rep.parl.lead` | Women's share of lower house seats (next election) | Women in Parliament Dataset |
| `gef` | Gender-targeted public funding (binary) | IDEA / custom coding |
| `gef.allocation` | Allocation-based GTPF mechanism | IDEA / custom coding |
| `gef.eligibility` | Eligibility-based GTPF mechanism | IDEA / custom coding |
| `gef.earmarking` | Earmarking-based GTPF mechanism | IDEA / custom coding |
| `v2elpubfin` | Public campaign finance (continuous) | V-Dem v15 |
| `vdem.poly` | Electoral democracy index | V-Dem v15 |

## Data

Three datasets are required (not included in this repository):

**Women in Parliament Dataset** (custom dataset, available on request)  
→ Place as `data/women_parliament.csv`

**V-Dem v15** — Varieties of Democracy country-year dataset  
Download: https://v-dem.net/data/the-v-dem-dataset/  
→ Place as `data/V-Dem-CY-Full+Others-v15.csv`

**POLCON 2025** — Political Constraints Dataset  
Download: https://mgmt.wharton.upenn.edu/faculty/heniszpolcon/polcondataset/  
→ Place as `data/POLCON_2025_FINALPOSTED.csv`

## Running the code

1. Clone this repository
2. Place the three data files in `data/`
3. Open `replication.R` and run from top to bottom

The script is structured in numbered sections (1–14) and can be run
section by section.
